# WordPress插件 - 百度SEO优化

完全免费全后台可视化配置，自动生成SEO友好Sitemap，百度快速收录，WordPress全站SEO优化，每天自动推送，伪原创设置。

## 功能特性

- ✅ 自动生成 SEO 优化的 Sitemap.xml
- ✅ 百度快速收录 API 推送
- ✅ 每天定时自动推送
- ✅ 发布/更新文章时自动推送
- ✅ 生成优化的 robots.txt
- ✅ 文章伪原创（仅对百度蜘蛛生效）
- ✅ 百度蜘蛛访问日志
- ✅ URL 去重和推送状态管理
- ✅ 全后台可视化配置
- ✅ 完全免费开源

## 安装方法

1. 下载插件压缩包
2. 登录 WordPress 后台
3. 进入"插件" -> "安装插件"
4. 点击"上传插件"
5. 选择下载的压缩包并安装
6. 启用插件

## 使用说明

1. 进入 WordPress 后台的"百度SEO优化"菜单
2. 在"插件配置"页面填写百度推送接口地址
3. 点击"生成SEO Sitemap并推送百度"测试推送功能
4. 可在"SEO优化设置"中配置更多选项
5. 可在"伪原创设置"中开启伪原创功能

## 百度接口地址格式

```
http://data.zz.baidu.com/urls?site=https://www.sgvps.cn&token=xxx
```

请替换 `https://www.sgvps.cn` 为您的网站域名，`xxx` 为您的百度推送 token。

## 系统要求

- WordPress 5.0 或更高版本
- PHP 7.2 或更高版本
- 建议启用 HTTPS
